ShellScript

Um script é basicamente um roteiro ou uma receita de procedimentos escrito em linguagem de computador e que executa diversas funções e/ou tarefas em um computador.

ShellScript

As linguagens de script são ferramentas utilizadas para controle de um determinado programa ou aplicativo; para configuração ou instalação em sistemas operacionais; entre outros controles de procedimentos.

ShellScript

Algumas linguagens de programação geralmente usadas como script são: JavaScript, Lua, PHP, Python, ShellScript, Ruby, VBScript, etc.

ShellScript

Sempre para iniciar o script de tarefas (shell script), precisamos usar o Shebang (#!), junto com /bin/bash (O terminal que estiver usando, o mais comum é o bash). Sempre na primeira linha. Então devemos sempre iniciar o script com #!/bin/bash. Esse comando serve para rodar seu script, chamando com o nome do seu terminal.

ShellScript - Como usar?

Para usar, precisamos criar um script, escolhendo um dos exemplos abaixo:

					    	
							$ sh ./nome_do_script.sh ou
							$ ./nome_do_script.sh ou 
							$ bash /caminho_completo/nome_do_script.sh

ShellScript - Revisando

  • Todo script deve começar com “#!/bin/bash”
  • Cada linha é um novo comando (Procedimento)
  • Linhas de comentário devem começar com um #
  • O comando echo serve para exibir um texto na tela. Ex: echo "Hello"

Esturura do ShellScript

No exemplo será utilizado o shell bash que a maioria das distribuições Linux utiliza de forma nativa o bash. Veja a seguir como fazer o famoso “Hello World” em shell script:

					
					#!/bin/bash

					# Meu primeiro script

					echo “Hello World!”

Gravando o Arquivo do ShellScript

Após digitar o texto, você deve salvar o script com a extensão .sh

						
						$ vi helloworld.sh

						Para executar:

						$ ./helloworld.sh

						Mas antes precisamos dar permissão de execução para o arquivo

						$ chmod +x helloworld.sh

outras Opções:

						
						#!/bin/ksh  

						# Meu primeiro script

						echo Hello World

outras Opções:

					
					#!/bin/sh  

					# Meu primeiro script

					echo Hello World

Um pouco de ShellScript

ShellScript é como uma linguagem de programação normal então vamos ver mais a fundo:

Variáveis em shell script

						
					variavel="valor"
					chamar a variavel com echo, Exemplo:

					$ var1 = "Estou criando uma variavel"
					$ echo $var1

					$ echo "Entre com o valor para a variável: " ;
					$ read var2
					$ echo var2

Controle de fluxo com o if

					
						if [ -e $linux ]
						then
						  echo 'A variável $linux existe.'
						else
						  echo 'A variável $linux não existe.'
						fi

Controle de fluxo com o if

					
						-eq	Igual
						-ne	Diferente
						-gt	Maior
						-lt	Menor
						-o	Ou
						-d	Se for um diretório
						-e	Se existir
						-z	Se estiver vazio
						-f	Se conter texto
						-o	Se o usuário for o dono
						-r	Se o arquivo pode ser lido
						-w	Se o arquivo pode ser alterado
						-x	Se o arquivo pode ser executado

if aninhado

					
						if [ -z $1 ]; then
						  echo "Você tem de entrar com um parâmetro válido"
						  exit
						elif [ $1 = "parametro1" ]; then
						  comando1
						  comando2
						elif [ $1 = "parametro2" ]; then
						  comando3
						  comando4
						else
						  echo "Você tem de entrar com um parâmetro válido"
						fi

case

					
						case $1 in
						  parametro1) comando1 ; comando2 ;;
						  parametro2) comando3 ; comando4 ;;
						  *) echo "Você tem de entrar com um parâmetro válido" ;;
						esac

for

					
						for i in 1 2 3 4 5
						do
						  echo "Random number $i: $RANDOM"
						done

while

					
						variavel="valor"
						while [ $variavel = "valor" ]; do
						  comando1
						  comando2
						done

until

					
						variavel="naovalor"
						until [ $variavel = "valor" ]; do
						  comando1
						  comando2
						done
